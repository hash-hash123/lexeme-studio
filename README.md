# Lexeme Studio

多言語語彙学習プラットフォームの MVP 実装です。  
要件書の「母語 × 学習言語」「語彙概念中心」「認識 / 想起カード」「基本 SRS 復習」を、React + TypeScript + Vite のローカル完結アプリとしてまとめています。

## 含まれる機能

- 日本語 / 英語 / ドイツ語の 6 学習ペア切替
- Lexeme / Sense / StudyItem ベースのデータモデル
- L0-L1 のシード語彙コース
- 認識カードと想起カードのレビュー
- 音声再生
- SRS 状態管理とローカル保存
- 語彙ブラウザの検索 / フィルタ
- 学習進捗の可視化
- 母語に応じた UI 文言切替

## 開発

```bash
npm install
npm run dev
```

ブラウザで `http://127.0.0.1:5173` を開いてください。

## 検証

```bash
npm run lint
npm run build
```

## 補足

- 進捗は `localStorage` に保存されます。
- バックエンドや認証はまだ入れていません。
- 次の拡張先として、実データ投入、Supabase 接続、例文穴埋め、音声アセット管理を想定しています。
